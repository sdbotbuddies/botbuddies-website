BotBuddies is a programming and robotics team for 6th and 7th graders who want to learn how real software is built. We use the same tools that professional developers use every day — VS Code, Python, Git, and GitHub.

Students work on coding assignments, personal projects, and FLL robot programming — all while building skills that will last a lifetime.
